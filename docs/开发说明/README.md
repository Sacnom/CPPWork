## 🎯 每份文档里都有什么

每份专属说明都包含：

1. **你要做什么** —— 用大白话解释你的模块功能
2. **先弄懂几个概念** —— 涉及的 C++ 知识点，零基础也能看懂
3. **完整代码** —— 可以直接复制粘贴，能编译通过的完整实现
4. **怎么测试** —— 一步一步教你验证代码是否正确
5. **容易踩的坑** —— 初学者常犯的错误和注意事项

## 🚀 快速开始流程

1. **打开你的专属文档**（上面表格里找到你的链接）
2. **读完「先弄懂几个概念」部分**（5 分钟）
3. **把「完整代码」复制到你负责的 `.cpp` 文件里**
4. **按「怎么测试」的步骤验证功能**
5. 有问题先看「容易踩的坑」，再问组里

## ⚠️ 重要提醒

- **只改你负责的那一个 `.cpp` 文件**，别动其他人的文件
- **不要随便改 `BookManager.h` 和 `Book.h`**，这俩是全组共享的
- **每次改完先编译测试**：`g++.exe -std=c++17 -g src/*.cpp -o build/main.exe`
- **提交前先 `git pull`**，避免冲突
